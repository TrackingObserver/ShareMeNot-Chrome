Changelog

ShareMeNot for Chrome

Version 1.2.1

Released on 2013-12-10 .

  • Reduced size of the LinkedIn replacement button.

Version 1.2

Released on 2012-12-30 .

  • Fixed issue that caused problems on Twitter itself.

Version 1.1.2

Released on 2012-11-11 .

  • Added an option to remove cookies from requests to trackers and load the real social buttons, rather than replacing the buttons entirely. This option is less privacy-preserving, but will not cause page layout issues.

Version 1.0.1

Released on 2012-07-06 .

  • Reduced the size of the Facebook replacement button and the +1 replacement button to prevent ShareMeNot from breaking page layouts.
  • Fixed issue in which attempting to block a tracker that was previously unblocked using the popup menu did not work.

Version 1.0

Released on 2012-06-25 .

  • Now out of beta.
  • Converted replacement buttons to SVG image files rather than HTML files embedded using iframe elements.
  • Improved appearance of the page action popup that opens when the ShareMeNot icon is clicked on a page with trackers.
  • Minor bug fixes.
  • Shares same code base as ShareMeNot for Firefox version 2.0.

Version 0.3 (beta)

Released on 2012-06-15 .

  • Requires Chrome 18 or higher.
  • Added Pinterest support.
  • The replacement buttons now automatically size to fit their contents.
  • Added options page.
  • Added ability to choose which trackers to block and which to not block.
  • Added ability to add exceptions so that ShareMeNot doesnâ€™t block certain trackers on certain web pages.
  • Updated icon.
  • Upgraded to use Chrome extension manifest version 2.
  • Improved security by restricting which extension resources can be loaded by external web pages.
